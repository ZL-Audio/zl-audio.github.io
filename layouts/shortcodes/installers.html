Unsure if your CPU supports AVX2?
For Windows: Check your CPU model in Task Manager (Ctrl+Shift+Esc → Performance → CPU):
  • Intel Core: i3/i5/i7/i9 from the 4th generation (Haswell, 2013) or newer. Modern Core Ultra and "i-less" Core models are fully supported. (Pentium and Celeron are typically not supported).
  • Intel Xeon: E5 v3 (Haswell-EP) or newer. Modern Xeon Scalable (Bronze/Silver/Gold/Platinum) are fully supported.
  • AMD Ryzen: All Ryzen processors (Zen architecture and newer) support AVX2. (Older AMD FX or Athlon processors are not supported).

If still unsure, check your CPU on CPU-Z or search the exact model name online.

For Linux: Run grep avx2 /proc/cpuinfo in your terminal. If it outputs text, AVX2 is supported.
