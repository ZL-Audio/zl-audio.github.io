Unsure if your CPU supports AVX2?
For Windows: Check your CPU model in Task Manager (Ctrl+Shift+Esc → Performance → CPU):
  • Intel: Must include the word Core. If it is an i3/i5/i7/i9, the generation number must be 4000 or higher (e.g., Core i7-4770K, Core i9-14900K). Newer Core Ultra and "i-less" Core models are fully supported (e.g., Core Ultra 7 265K, Core 5 120U).
  • AMD: Must include the word Ryzen. All Ryzen processors support AVX2.
For Linux: Run grep avx2 /proc/cpuinfo in your terminal. If it outputs text, AVX2 is supported.
