## 卸载指南

以下指南将帮助您彻底卸载 ZL Audio 插件。很遗憾看到您离开，但非常感谢您试用 ZL Audio 的产品。

> 如果您曾手动将插件二进制文件（后缀为 `.vst3`、`.lv2` 等的文件）移动到了其他位置，请前往您自定义的文件夹进行删除。

## macOS 卸载

1. 删除插件二进制文件

前往以下文件夹，删除对应的 `.vst3`、`.component` 或 `.aaxplugin` 文件：

- VST3: `/Library/Audio/Plug-Ins/VST3`
- AU: `/Library/Audio/Plug-Ins/Components`
- AAX: `/Library/Application Support/Avid/Audio/Plug-Ins`

2. 删除 UI 界面文件

前往 `~/Library/ZL Audio`，删除对应的子文件夹或整个文件夹。

3. 删除其他预设文件

前往 `~/Library/Audio/Presets/ZL`，删除对应的子文件夹或整个文件夹。

## Windows 卸载

1. 删除插件二进制文件
前往以下文件夹，删除对应的 `.vst3`、`.lv2` 或 `.aaxplugin` 文件：

- VST3: `C:\Program Files\Common Files\VST3`
- LV2: `C:\Program Files\Common Files\LV2`
- AAX: `C:\Program Files\Common Files\Avid\Audio\Plug-Ins`

2. 删除 UI 界面文件

前往 `C:\Users\[您的用户名]\AppData\Roaming\ZL Audio`，删除对应的子文件夹或整个文件夹。

3. 删除其他预设文件

前往 `C:\Users\[您的用户名]\AppData\Roaming\Audio\Presets\ZL`，删除对应的子文件夹或整个文件夹。

## Linux 卸载

1. 删除插件二进制文件

前往您的安装目录，删除对应的 `.vst3` 或 `.lv2` 文件。

2. 删除 UI 界面文件

前往 `~/ZL Audio`，删除对应的子文件夹或整个文件夹。

3. 删除其他预设文件

前往 `~/Audio/Presets/ZL`，删除对应的子文件夹或整个文件夹。
