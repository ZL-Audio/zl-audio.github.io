The following guidance helps you completely uninstall ZL Audio plugins. We are sorry to see you go, but we appreciate you giving ZL Audio plugins a try.

> If you have manually moved the plugin binaries (files ending in `.vst3`, `.lv2`, etc.) to a different location, you will need to locate and delete them from that specific folder instead.

## macOS Uninstallation

1. Delete plugin binaries

Navigate to the following folders and delete the `.vst3`, `.component`, or `.aaxplugin` files:

* VST3: `/Library/Audio/Plug-Ins/VST3`
* AU: `/Library/Audio/Plug-Ins/Components`
* AAX: `/Library/Application Support/Avid/Audio/Plug-Ins`

2. Delete UI files:

Navigate to `~/Library/ZL Audio` and delete the corresponding sub-folder or the whole folder.

3. Delete Other Presets:

Navigate to `~/Library/Audio/Presets/ZL` and delete the corresponding sub-folder or the whole folder.


## Windows Uninstallation

1. Delete plugin binaries

Navigate to the following folders and delete the `.vst3`, `.lv2`, or `.aaxplugin` files:

* VST3: `C:\Program Files\Common Files\VST3`
* LV2: `C:\Program Files\Common Files\LV2`
* AAX: `C:\Program Files\Common Files\Avid\Audio\Plug-Ins`

2. Delete UI files:

Navigate to `C:\Users\[YourName]\AppData\Roaming\ZL Audio` and delete the corresponding sub-folder or the whole folder.

3. Delete Other Presets:

Navigate to `C:\Users\[YourName]\AppData\Roaming\Audio\Presets\ZL` and delete the corresponding sub-folder or the whole folder.
